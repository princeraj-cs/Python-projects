# Modern Quiz Game GUI

## Features
- **Beautiful Modern Interface** - Clean, professional design using PyQt5
- **Progress Tracking** - Visual progress bar showing quiz completion
- **Live Score Display** - Real-time score updates
- **Instant Feedback** - See if you're correct or incorrect immediately
- **Question Counter** - Know exactly which question you're on
- **Completion Summary** - Get your final score and percentage

## Installation

Before running the GUI, you need to install PyQt5:

```bash
pip install PyQt5
```

## Running the Quiz

Simply run the main.py file:

```bash
python main.py
```

Or directly run the GUI:

```bash
python quiz_gui.py
```

## How to Use

1. Read the question carefully
2. Click either **TRUE** or **FALSE** button to answer
3. Receive instant feedback showing if you were correct
4. Click **Next Question** to proceed
5. After all questions, you'll see your final score and percentage

## UI Components

- **Title Bar** - Shows "Quiz Master" and your current score
- **Progress Bar** - Visual representation of quiz progress
- **Question Display** - Large, clear question text
- **Answer Buttons** - Green (TRUE) and Red (FALSE) buttons
- **Feedback Area** - Shows correct/incorrect feedback
- **Next Button** - Proceeds to the next question

## Customization

You can customize the appearance by editing the color codes in `quiz_gui.py`:
- `#27ae60` - Green (correct color)
- `#e74c3c` - Red (incorrect color)
- `#3498db` - Blue (primary color)
- `#2c3e50` - Dark blue (text color)
